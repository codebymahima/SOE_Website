# 🏫 School of Engineering (SOE) Website

A responsive frontend website for the **School of Engineering, Jawaharlal Nehru University (JNU)** built using **React**, **Vite**, and **Tailwind CSS**. The project recreates the official university website with a cleaner UI, reusable React components, and responsive design principles.

---
## Live demo
https://soe-website-sable.vercel.app/

## 🚀 Features

* Responsive and modern user interface
* Component-based architecture
* React Router for seamless page navigation
* Clean institutional design inspired by the official SOE website
* Reusable UI components
* Fully responsive navigation bar
* Homepage with welcome section and banner
* Programs, Faculty, Research, Events, and Contact pages
* Contact form UI
* Interactive cards for programs, faculty, events, and notices
* Organized project structure for scalability

## ⚙️ Installation

```bash

cd SOE website
npm install
npm run dev
```
